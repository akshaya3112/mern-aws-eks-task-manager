# 🚀 TaskFlow – MERN Task Management System

A modern full-stack task management application built using the MERN stack with secure authentication, Docker, Kubernetes deployment, and responsive UI.

## 🌐 Live Demo

Frontend:
https://YOUR-VERCEL-LINK.vercel.app

Backend API:
https://mern-aws-eks-task-manager.onrender.com

## ✨ Features

- JWT Authentication
- User Registration & Login
- Create Tasks
- Edit Tasks
- Delete Tasks
- Mark Complete
- Dark / Light Mode
- Responsive Design
- Modern UI

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- CSS Modules
- Axios

### Backend
- Node.js
- Express.js
- JWT Authentication

### Database
- MongoDB
- MongoDB Atlas

### DevOps & Deployment
- Docker
- Kubernetes
- AWS EKS
- Render
- Vercel
